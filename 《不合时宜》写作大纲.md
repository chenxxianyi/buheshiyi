## 一、书名与核心精神

**书名出处：** 苏轼自嘲——「一肚皮不合时宜」。既是人物的命运注脚，也是整个故事的核心精神。

**一句话概括：** 一群「不合时宜」的人，在一个容不下异见的时代里，各自寻找自己的答案。

**基调：** 沉郁中见风骨，悬疑中见人心。不写快意恩仇，写「不合时宜」之人的选择。

---

## 二、中心思想

### 总主题

在一个站队比站对更重要的时代，坚持说真话的人能活多久？——小说不直接回答，但通过人物的选择来呈现。

### 三个核心母题

#### 1. 风骨与生存

| 人物 | 困境 | 回答 |
|------|------|------|
| 沈惊澜 | 保命还是保良心？他只是不小心碰到了不该碰的东西 | 从藏锋不出的聪明人，到自己选的傻子 |
| 苏轼 | 不站党，只站对。代价是三次被贬，半生流放 | 「此心安处是吾乡」 |

**时代提问：** 在一个站队比站对更重要的时代，坚持说真话的人能活多久？

#### 2. 真相的代价

| 人物 | 困境 | 回答 |
|------|------|------|
| 欧阳听霜 | 知道真相之后，还能回到原来的生活吗？ | 不能。但总得有人知道 |
| 欧阳修（已故） | 有些事，知道的人越少越安全 | 「但总得有人知道」 |

**时代提问：** 有些真相知道了就要死。你还想知道吗？

#### 3. 朝堂与江湖的张力

- 朝廷借新政扩张权力，触角伸向江湖
- 江湖不是法外之地——门派有田产、有佃户、有生意，新政每一项都动其根本
- 有人在朝堂上斗，有人在江湖上杀——两条战线，同一场战争

**核心张力：** 朝廷要秩序，江湖要自由。但朝廷内部不团结，江湖内部也不团结。真正的争斗不在朝廷与江湖之间，而在「想控制一切的人」与「想留一点余地的人」之间。

### 隐含的「时代之问」

小说不直接回答这些问题，但通过人物的选择来呈现：

| 问题 | 在谁身上体现 |
|------|-------------|
| 一个人可以不属于任何阵营吗？ | 苏轼 |
| 忠义与真相冲突时，选哪个？ | 沈惊澜 |
| 复仇和真相是一回事吗？ | 欧阳听霜 |
| 善良的制度为什么变成吃人的工具？ | 王安石（侧写） |
| 知道秘密的人，有责任说出来吗？ | 欧阳修（已故） |

## 三、故事世界

### 历史背景

| 要素 | 内容 |
|------|------|
| 时代 | 北宋·神宗熙宁年间（核心：熙宁三年至七年，约 1070–1074） |
| 历史事件 | 王安石变法、新旧党争、苏轼被贬 |
| 真实人物 | 苏轼、欧阳修、王安石、吕惠卿、宋神宗 |
| 虚构人物 | 沈惊澜（男主）、欧阳听霜（女主）及江湖群像 |
| 类型 | 历史武侠 · 权谋悬疑 · 爱情 |

### 真实时间线与虚构叠加层

| 年份 | 历史事件 | 小说中的位置 |
|------|---------|-------------|
| 1069（熙宁二年） | 王安石拜参知政事，开始变法 | 故事起点。青苗法、保甲法陆续推行 |
| 1070（熙宁三年） | 旧党全面反弹。司马光退居洛阳 | 欧阳修秘密会面（虚构），获悉危险 |
| 1071（熙宁四年） | 苏轼上书反对新法，触怒王安石 | 苏轼被外放杭州（历史真实：通判杭州） |
| 1072（熙宁五年） | 欧阳修病逝于颍州（历史真实） | 小说暗线：并非自然死亡 |
| 1074（熙宁七年） | 王安石第一次罢相 | 故事收束点。吕惠卿失势 |

在真实历史之上，小说构建一层「隐藏的真实」：

| 历史表象 | 小说隐藏层 |
|----------|-----------|
| 欧阳修病逝 | 被慢性毒杀（知道得太多了） |
| 苏轼被贬 | 贬谪途中有人安排暗杀 |
| 吕惠卿争权 | 他的背后还有更大的推手 |
| 新法推行 | 某些条款被利用来清洗江湖势力 |
| 皇城司 | 内部被渗透，部分密探在执行「非皇命」的任务 |

## 四、人物谱系

### 4.1 主角

#### 沈惊澜（男主）

| 维度 | 设定 |
|------|------|
| 年龄 | 二十七岁 |
| 身份 | 枢密院·检详房，正八品检详官 |
| 日常 | 整理边防奏报、归档机密文书。管档案的，不是拿刀上阵的武将 |
| 出身 | 父亲是定州路兵马钤辖，战死于西夏前线。以荫补入仕，没有科举功名 |
| 处境 | 品级低但经手机密。有才能但不愿站队，上司不提拔他。在枢密院五年，原地踏步 |
| 性格 | 外表：斯文寡言，眉目温和，像账房先生。内里：固执、认死理、对就是对错就是错——这一点和苏轼一模一样，只是他不写诗 |
| 武功 | 家传枪法（父亲教的实战枪，不是花枪）。在禁军中也练过弓马。武功不算顶尖，但胜在实战经验——他见过战场，知道什么叫拼命 |
| 对苏轼的态度 | 素未谋面。但读过苏轼的《教战守》，其中对兵制的批评和他在枢密院看到的积弊完全吻合。他对苏轼是「神交」——一个陌生人说中了自己的心里话 |
| 核心驱动 | 一开始是自救（看到了不该看的东西，要被灭口）；后来变成责任（既然已经卷进来了，不能看着一个好人被杀） |
| 弧线方向 | 藏锋不出的聪明人 → 自己选的傻子。从「不卷入」到「主动承担」。从聪明人变成了傻子——但这是他自己选的 |

**沈惊澜之父：沈恪**

| 维度 | 设定 |
|------|------|
| 生卒 | 约 1018–1068，卒年约五十 |
| 官职 | 定州路兵马钤辖，正五品，常年驻守河北边防 |
| 性格 | 沉默、规矩、不打折扣的职业军人。不结党、不走关系、不屑逢迎。一辈子只做一件事：守住他该守的地方。对儿子的爱不说出口，全在枪法里 |
| 与军饷案的关系 | 庆历三年（1043），三十出头的沈恪是定州前线接收军饷的三名签字军官之一。他经手的是封好的箱子——他只管点数签收，不知箱中实际数额。二十年后，这个签名成了他的催命符 |
| 死因 | 熙宁元年（1068），西夏犯边。沈恪被调往突出部驻防——一个「明知守不住」的位置，援军迟迟不到。他在一场本不该打的遭遇战中阵亡。事后军方奏报：「力战殉国」。真相：借西夏之手灭口 |
| 留给沈惊澜的东西 | 一杆铁枪、四字枪诀（刺、挡、挑、扫）、一封最后的信。信上只有一句话：「做好你的事。」——朴实到残忍。沈惊澜后来才明白：父亲知道自己会死，但没有逃。他不逃，是因为逃了就不叫沈恪 |
| 在小说中的作用 | 生前是影子——沈惊澜的性格源头。死后是线索——他的名字出现在欧阳修手稿中，成为卷三沈惊澜觉醒的关键：「这不是别人的事，从一开始就不是」 |

#### 欧阳听霜（女主）

| 维度 | 设定 |
|------|------|
| 年龄 | 二十二岁 |
| 身份 | 欧阳修三子欧阳棐之女。祖父去世后，寡居汴京旧宅 |
| 出身 | 欧阳修是仁宗、英宗、神宗三朝元老，文坛领袖。但祖父去世后，欧阳家已经不再显赫——父亲欧阳棐只是普通州官，门庭渐冷 |
| 性格 | 外表是书香门第的大小姐——温文、知礼、说话轻声。内里继承了祖父的风骨：刚、直、能忍但绝不让。她比沈惊澜更沉，但也更独——不习惯依靠任何人 |
| 武功 | 祖父晚年隐居颍州时，一位少林俗家高手因敬重欧阳修人品，自愿教她武艺。学的是少林一脉的扎实功夫——拳掌为主，短刃为辅。不是轻灵的「女侠路数」，而是硬桥硬马，不花哨 |
| 对苏轼的态度 | 复杂。苏轼是祖父最疼的学生，她小时候叫他「苏大哥」。但祖父死后，苏轼没来吊唁（历史真实：苏轼确实因政治原因未能亲至）。她心里有结——不是怨，是「你是不是也知道什么但没说」 |
| 核心驱动 | 追问祖父死因。收到匿名信后，她不能假装没看见。她要找到苏轼，问一个问题——「祖父临死前，跟你说了什么？」 |
| 弧线方向 | 一个人扛 → 允许有人并肩。从「我不需要别人」到「她没有拒绝」——这是她最大的成长 |

### 4.2 历史人物（小说中的定位）

#### 苏轼（核心历史人物，非 POV）

| 维度 | 设定 |
|------|------|
| 在小说中的角色 | 不是主角，也不是导师。是线索的终点——两个主角要找到他，但目的不同。他是风暴中心，但风暴不是他制造的 |
| 出场方式 | 前期：侧面描写（沈惊澜读他的文章、同僚谈论他、官差押解他）。中期：实际出场（杭州见面）。他不是「智者指引」，而是一个有血有肉的、身处困境的人 |
| 人物刻画要点 | 1. 幽默感——再难的时候也能说笑话。2. 不装——在农民面前和皇帝面前说话一样。3. 固执——他反对王安石的激进，也反对司马光的全盘推翻，两边不讨好 |

#### 欧阳修（已故，贯穿全书的影子）

| 维度 | 设定 |
|------|------|
| 功能 | 小说最大的「失踪人物」。他不在了，但他的影子压在每一个人身上 |
| 隐藏设定 | 晚年曾秘密调查仁宗朝一桩旧案——涉及先帝、宦官、一笔巨额军饷的去向。他没有完成调查就「病」了。他死前把关键证据藏了起来 |
| 与各人物的关系 | 对欧阳听霜：祖父的疼爱 + 刻意的保护（不告诉她真相）。对苏轼：有知遇之恩，但也留下了苏轼不知情的秘密。对沈惊澜：沈的父亲可能和欧阳修的调查有关联 |

#### 吕惠卿（主要反派）

| 维度 | 设定 |
|------|------|
| 历史真实 | 王安石变法的主要助手，精明强干但权欲熏心。后来与王安石反目，被贬 |
| 小说中的定位 | 中层反派。不是最终推手，但他是最危险的执行者。在苏轼贬谪途中安排暗杀 |
| 塑造要点 | 不是脸谱化的坏人。真心相信变法能强国，只不过认为清除障碍是必要的代价。聪明、高效、毫无道德负担——最危险的那种人 |

#### 王安石（侧写人物）

**已确认：卷一有一场侧面出场。**

沈惊澜在枢密院廊下偶遇王安石及其随员从议政堂退出。王一边疾走一边向书吏口授政令，目不斜视。一名年轻官员跑着追他，手中文书散落一地——王停下步子，弯腰替他捡起，递还后一言不发继续走。沈惊澜退到墙边让他们通过。两人目光对视了一瞬——锐利、焦灼、不容分说——然后是背影。

这幕的功能：不用对话，用一组动作写出王安石的全部——高效、勤政、不近人情、不觉得自己需要为任何代价停下来。也写出沈惊澜的「看」：他看见了这个人身上的力量，也看见了危险。

- 定位：新政的灵魂，但不是阴谋的参与者。
- 隐含态度：他大概知道底下的人在做什么，但他选择不问。
- 「以天下为己任」的背面是「不计代价」——这是小说对王安石的含蓄批评。

#### 赵宗谔（最终推手）

| 维度 | 设定 |
|------|------|
| 身份 | 安化郡王。仁宗堂弟、周王赵元俨之孙。神宗朝宗正寺卿，掌皇族事务 |
| 年龄 | 约五十五岁（熙宁年间） |
| 与军饷案的关系 | 庆历年间，他以皇亲身份监领河北军需，是军饷秘密调拨的实际经手人。五十万贯中有相当部分经由他手流入河朔民间武装的扶植计划——这计划本有先帝默许，但账目不清，经不起查。他是这件事唯一的活着的责任人 |
| 动机 | 不是贪财。是保住秘密。军饷案一旦曝光，牵出的不是个人贪腐，而是仁宗朝皇室暗中豢养非官武装的政治丑闻——足以动摇赵宋宗室的根基。他自认在「护国」：清理所有知情人，不是作恶，是止损 |
| 手段 | 不亲自下场。通过吕惠卿调度皇城司（部分已渗透），以「清除新法障碍」的名义执行暗杀和监视。吕惠卿以为自己在利用赵宗谔的权力——事实正相反 |
| 在小说中的位置 | 全书从来不以他的 POV 展开。他的名字直到卷三才第一次出现。全书他只在一场戏里正面出场——神宗召见宗室时，他与沈惊澜有一次极短的对视。他看上去是一位温和、有礼、无可挑剔的长者——这让真相更加寒冷 |
| 结局 | 吕惠卿倒台后，他全身而退。姓赵的人，总是安全的。小说不给他「恶有恶报」的结局——这是「不合时宜」最冷的一刀 |

## 五、总体结构

### 四卷概览

| 卷 | 主题词 | 核心事件 | 功能 |
|----|--------|---------|------|
| **卷一·暗流** | 触碰 | 两个人各自碰到了不该碰的东西 | 悬疑铺设。建立人物与氛围。最慢 |
| **卷二·追途** | 同行 | 两个陌生人被迫结伴，在追杀中南下，沿途拼凑线索 | 追杀与追查并行。加速 |
| **卷三·白夜** | 真相 | 抵达杭州，所有线索汇聚。见到苏轼，揭开秘密 | 高潮。真相汇聚，决战 |
| **卷四·余声** | 选择 | 真相大白，扳倒吕惠卿（符合历史真实）。各人走向自己的归宿 | 回落。给人物一个去处（不一定是圆满） |

### 叙事视角

双 POV，交替切换。奇数章沈惊澜、偶数章欧阳听霜。合流后（卷三起）不再严格区分，以场景需要为准。

### 空间哲学

| 地点 | 象征 |
|------|------|
| **汴京** | 权力中心，压抑、危险、每个人都被监视 |
| **颍州** | 过渡地带，欧阳修的归隐地，真相的入口 |
| **镇江** | 宗教与世俗的交汇点，「看破」的象征 |
| **杭州** | 边缘与中心之间——远离汴京但仍是朝廷管辖。在这里，远离权力反而离真相更近 |

从汴京到杭州，不仅是地理上的南下，也是从「暗流」到「真相」的推进。

### 关键地点

| 地点 | 功能 |
|------|------|
| 汴京·枢密院 | 沈惊澜工作地，密信被发现之处。冰冷、肃穆、布满暗影的官衙 |
| 汴京·欧阳旧宅 | 欧阳修生前居所。人去楼未空——书房里藏着秘密，院外有人监视 |
| 颍州·欧阳旧居 | 欧阳修归隐终老之地。荒草丛生，书架暗格藏有三封信。第一次大截杀的战场 |
| 镇江·金山寺 | 苏轼夜宿之地（历史真实）。无尘和尚隐居后山。关键信息「东西在杭州」在此获得 |
| 杭州·西泠桥 | 桥畔第三株柳树下，埋着欧阳修的铜箱。西湖边的秘密 |
| 杭州·西湖畔 | 最终暗战战场。湖光山色之间，两个时代的弃儿与朝廷暗探决一死战 |
| 颍州·听风揽月馆 | 尾声：沈惊澜与欧阳听霜所开的小武馆。不立招牌，不收大钱。全书最后的画面 |

## 六、三层悬念

### 第一层（读者最先看到）

**谁要杀苏轼？为什么要杀？**

→ 答案：吕惠卿（卷三揭晓）

### 第二层（中期浮现）

**欧阳修是怎么死的？**

→ 答案：被灭口。他查到了不该查的东西（卷三揭晓）

### 第三层（最终谜底）

**这一切的根源是什么？**

→ 答案：仁宗朝的秘密军饷案。一个跨越二十年的秘密。仁宗庆历年间，西北用兵时有一笔五十万贯的军饷消失。经办此事的五个人后来都死了。欧阳修追查此事时被发现。有人现在要借新政把尾巴彻底割掉——包括所有知道旧事的人。（卷三中后段揭晓）

## 七、感情线

### 基调

七分隐忍，三分对抗。感情不是小说的主线，但它是人物弧线的必要支撑——两个人都是在「选择信任另一个人」的过程中，完成了自己的成长。

### 四个阶段

| 阶段 | 关系状态 |
|------|---------|
| **敌意** | 互相以为对方是刺客。第一印象是警惕 |
| **试探** | 被迫同行。交换信息但并不交心。各有保留 |
| **裂痕与转折** | 他挡箭（她看到他肯替她死）。她帮他取解药（他看到她的果断和可怕）。彼此开始重新评估对方 |
| **默契** | 不再互相试探。从「你和我」变成「我们」。最后的表白不是「我爱你」，而是「我送你」「反正我也没别的地方可去」 |

### 感情线不做什么

- ❌ 没有一见钟情
- ❌ 没有「英雄救美」的桥段（挡箭不是英雄救美，是两个人都在拼命）
- ❌ 没有三角恋、没有误会超过三章的分离
- ❌ 没有「在一起」的明确结局（留白的、在路上式的开放式结局）

## 八、写作原则

### 语言风格

| 方向 | 描述 |
|------|------|
| 叙述 | 简洁、节制。不堆砌形容词，不铺陈情绪——让动作和对话推进一切 |
| 对话 | 人物各自有口吻。苏轼有文气但不说文言；沈惊澜话少且不爱解释；欧阳听霜说话点到即止，但尖锐时不留情 |
| 打斗 | 快、准、有节奏。一场战斗通常不超过 500 字。重点不是招式描述，而是身体感受和结果 |
| 氛围 | 以环境写情绪，不直接写内心。雪、水、月、桥——用自然景物承载人物的孤独和温暖 |

### 历史严谨度

| 方面 | 标准 |
|------|------|
| 历史事件时间 | 准确。熙宁年号、苏轼贬谪年份、欧阳修卒年均按真实历史 |
| 历史人物性格 | 半真实。苏轼的幽默、欧阳修的风骨、吕惠卿的精明均取自史实，但赋予小说化的动机和行动 |
| 官职制度 | 准确。枢密院、检详房、皇城司、判官告院、通判等均为真实宋代官制 |
| 地理交通 | 准确。汴京→颍州→镇江→杭州的路线符合宋代实际驿路 |
| 日常生活 | 准确。饮食器皿、衣着称谓、货币单位均参考宋代笔记文献 |

### 写作红线

**✅ 可以写：**
- 政治阴谋中的暗杀、追杀、灭口
- 官场黑暗、制度缺陷
- 人物的复杂动机（非黑白分明）
- 轻功作为身体技艺（飞檐走壁、水面踏行）

**❌ 不要写：**
- 内功真气、以气驭剑、隔空伤人
- 「吃丹药暴涨功力」「奇遇得神功」等桥段
- 正面人物不经思考就「热血上头」
- 反派纯粹的恶（每个反派都有其逻辑和动机）
- 对王安石本人的直接丑化（他是悲剧性的矛盾人物，不是单纯的坏人）
